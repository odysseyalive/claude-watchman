# manifest.txt — the shipped file list for claude-watchman.
#
# install.sh fetches every path below from the repo (raw.githubusercontent) into
# the install directory — NO git clone required. Re-running the installer
# re-fetches them, so install and update are one and the same.
#
# Format: an optional flag, then the repo-relative path.
#   (none)  overwrite on every fetch — this is the portable product.
#   hook    chmod +x after fetch (executables).
#   keep    fetch only if absent — preserve a local copy (not used yet).
#
# NOT listed here (so a re-fetch never clobbers them): the machine-specific,
# gitignored artifacts — .env, config/watchman.conf, journal/findings.db,
# .claude/, CLAUDE.md — and .gitignore (install.sh appends its block idempotently).
# `watchman update --check` asserts this list stays in lockstep with the product.
